# StartTrackerPart2
This is an StatTracker APP that I created that uses PostGresSQl and the ORM Sequelize for users that can CRUD personal activities, statistics, lists. 


Tech this APP uses:
  - NodeJs
  - Express Generator
  - Clientside Javascript
  - Jade (Pug)Templating
  - PostgresSQL
  - Sequelize
  - CRUD
  - RestFul API's
  - Passport Authentication
  - Heroku Platform
